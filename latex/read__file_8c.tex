Adds a new line to the file. 

This function increments the heightfile count, allocates a new array to accommodate the additional line, and copies existing lines into this new array. The new line is then added to the end of the array. The old array is freed, and thetfile pointer is updated to the new array.


\begin{DoxyParams}{Parameters}
{\em game} & Pointer to the structure containing thetfile and heightfile information. \\
\hline
{\em line} & The new line to be added. This should be a valid string. If {\ttfamily NULL}, no action is taken, and the function returns {\ttfamily 0}.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
1 if the line was successfully added and thetfile was updated, 0 if the provided line was {\ttfamily NULL} or memory allocation failed.

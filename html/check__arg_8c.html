int check_walls ( t_data * m)

Checks if the map is correctly enclosed by walls on all sides.

This function checks the first and last rows of the map to ensure they are fully enclosed by '1's and also checks the first and last columns using the vertical_wall function. It reports an error if the map is not enclosed properly.

Parameters
mA pointer to the t_data structure that contains the game map and its dimensions.
Returns
Returns 0 if all walls are correctly enclosed, otherwise returns a non-zero value and reports an error.

Checks if all characters in a specific vertical wall of the map are '1'.

This function iterates through a column of the game map to verify if all characters are '1'. It is used to ensure that the map is correctly enclosed by vertical walls.

Parameters
gameA pointer to the t_data structure that contains the game map and its dimensions.
jThe column index to check for a vertical wall.
Returns
Returns 0 if the entire column is filled with '1's, otherwise returns 1.

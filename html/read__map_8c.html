int fill_map ( t_data *  m)

Fills the map in the t_data structure by modifying each row using the ones function.

Iterates over each row of the map (m->map) and calls the ones function on each row. The width of the map (m->widthmap) is used to specify the maximum number of characters to process for each row.

Parameters
mPointer to a t_data structure that contains the map data to be modified.

Calculates the longest width of atfile represented by a string.

Parameters

Returns
The width of thetfile as an integer.

Modifies the input string src by replacing certain characters with '1'.

Allocates a new string temp of size size + 2. It initializes this new string with '1' characters and then copies characters from src until it encounters a newline character, a null terminator, or exceeds the specified size. It replaces 'P' or '0' characters with the corresponding characters from src.

Parameters
srcThe source string to be modified. This pointer will be freed and replaced with a new allocated string.
sizeThe maximum number of characters to copy from src.

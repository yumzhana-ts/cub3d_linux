static int add_line ( t_data * game,
char * line )
static

Adds a new line to the heightmap.

This function increments the heightmap count, allocates a new array to accommodate the additional line, and copies existing lines into this new array. The new line is then added to the end of the array. The old array is freed, and the map pointer is updated to the new array.

Parameters
gamePointer to the structure containing the map and heightmap information.
lineThe new line to be added. This should be a valid string. If NULL, no action is taken, and the function returns 0.
Returns
1 if the line was successfully added and the map was updated, 0 if the provided line was NULL or memory allocation failed.

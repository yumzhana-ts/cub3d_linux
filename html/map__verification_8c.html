int validate_map ( t_data *  m)

Validates the game map and player position.

This function performs various checks to validate the map and player configuration. It checks if the map walls are correct, ensures the player has been placed properly, and verifies that the player is not in an invalid or locked position.

Parameters
mPointer to a t_data structure containing map and player data.
Returns
0 if the map is valid, 1 if there are wall issues, or an error code if validation fails.
